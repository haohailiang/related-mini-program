# Related Files
获取超哥feather框架相关联的文件

## Usage

按下`cmd+k cmd+v`快捷键呼出关联的文件列表

### Example of custom settings:
支持的文件列表如下
#### 组1
- `src/todos/components/visible-todolist/index.tsx`
- `src/todos/components/visible-todolist/scss/index.scss`
- `src/todos/container/visible-todo-list/index.ts`
- `src/todos/reducers/visibleTodoList.ts`
- `src/todos/actions/actionCreator.ts`

#### 组2
- `src/todos/actions/actionCreator.ts`
- `src/todos/actions/actionTypes.ts`
- `src/todos/reducers/index.ts`
- `src/todos/reducers/todoFilter.ts`
- `src/todos/reducers/todoList.ts`

#### 组3

- `src/login/components/login/index.tsx` 

- `src/components/error/lang/index.ts`
